Diseño web en 4 minutos

por Jeremy Thomas

Pongamos que tienes un producto, un portafolio, o simplemente una idea que quieres compartrir con todo el mundo en tu propio sitio web. Antes de publicarlo en internet, quieres darle un aspecto atractivo, profesional o, como mínimo, decente.

¿Qué es lo primero con lo que te tienes que poner?

Contenido

El objetivo del diseño es mejorar la presentación del contenido al que se aplica. Puede parecer obvio, pero dado que el contenido es el principal elemento de un sitio web, no debería quedar como algo para el final.

El contenido textual, como el párrafo que estás leyendo, es más del 90% de la Web. Aplicar estilos a este contenido hace muchísimo.

Supongamos que ya has acabado con el contenido que quieres publicar y has creado un fichero style.css vacío. ¿Cuál es la primera regla que puedes escribir?

Centrado

Las líneas largas de texto pueden ser difíciles de seguir, y por tanto difíciles de leer. EStablecer un límite de caracteres por línea mejora enormemente la legibilidad y el atractivo de un bloque de texto.

body {
  margin: 0 auto;
  max-width: 50em;
}

Después de dar estilo a los bloques de texto, ¿y si damos estilo al propio texto?

"Font family"

La fuente por defecto del navegador es "Times", que puede no resultar atractiva (sobre todo porque es la fuente "sin estilo"·"). Cambiar a una fuente sin serifas como "Helvetica" o "Arial" puede mejorar mucho el aspecto de la página.

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

Si prefieres quedarte con una tipografía con serifas, prueba con "Georgia".

Aunque hace el texto más atractivo, hagámoslo más legible también.

espaciado

Cuando al usuario una página le parece "rota", suele ser un problema de espaciado. Dar espacio tanto alrededor como dentro del contenido suele mejorar el atractivo de la página.

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

Aunque la disposición ha mejorado mucho hasta aquí, apliquemos más cambios sutiles.

Color & contraste

El texto negro sobre fondo blanco puede resultar agresivo a la vista. Elegir un tono más suave de negro para el texto principal hace la página más cómoda de leer.

body {
  color: #555;
}

Y para mantener un nivel aceptable de contraste, elijamos un tono más oscuro para las palabras importantes.

h1,
h2,
strong {
  color: #333;
}

Aunque la mayoría de la página ha mejorado visualmente, algunos elementos (como los fragmentos de código) siguen pareciendo fuera de lugar.

Balance

Sólo necesitamos algunos toques adicionales para corregir el balance de la página:

code,
pre {
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

Llegados aquí, a lo mejor te gustaría hacer que la página destaque y darle identidad.

Color primario

La mayoría de marcas tiene un color primario fuerte que le da un acento visual. En un sitio web este acento puede usarse para dar énfasis a los elementos interactivos, como los enlaces.

a {
  color: #e81c4f;
}

Pero para mantener el balance necesitaremos algunos colores adicionales.

Colores secundarios

El color de acento puede complementarse con tonos más sutiles para los bordes, fondos o incluso el texto principal.

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

Una vez cambiados los tonos, ¿por qué no cambiar las formas?

Tipografía personalizada

Dado que el texto es el principal contenido de una página web, usar una tipografía personalizada le da a la página una identidad aun más notable.

Puedes incrustar tu propia tipografía web o usar un servicio como Typekit, pero usaremose "Roboto", del servicio gratuito Google Fonts:

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto" "Helvetica", "Arial", sans-serif;
}

Después de mejorar la identidad a través del texto, ¿qué tal si añadimos mil palabras?

Se pueden usar gráficos e iconos tanto ornamentalmente para apoyar el contenido como tomando un rol principal en el mensaje que quieres comunicar.

Mejoremos nuestra cabecera con una bonita imagen de fondo obtenida de Unsplash

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

Añadamos también un logo.

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

Y aprovechemos la oportunidad para mejorar los estilos para el texto.

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

Y voilà!

Hemos diseñado una página decente en cuestión de minutos, siguiendo los principios básicos del diseño web. Sólo queda una última cosa por hacer...
